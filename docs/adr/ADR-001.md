# Evidence is the Core Domain Concept

## Context
HumanOS needs to support many types of personal data without becoming fragmented.

## Decision
All user-recorded activities are modeled as evidence.
Different activity types extend or specialize the Evidence concept.

## Consequences
Positive:
- Unified analytics
- Consistent event processing
- Easier integrations
- Cleaner extensibility

Trade-offs:
- Slightly more abstract than designing each feature independently.
- Requires careful domain boundaries to avoid an oversized "God entity."
